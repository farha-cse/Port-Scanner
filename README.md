# Python Port Scanner

## Description
A simple Python Port Scanner built using Python and the socket module.

## Features
- Scans ports 1–100
- Detects open ports
- Uses Python socket programming

## Technologies Used
- Python
- Socket Module

## How to Run

```bash
python3 port_scanner.py
```

Target:

scanme.nmap.org
